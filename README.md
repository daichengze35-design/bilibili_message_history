# bilibili message history

This project uses Selenium to scrape private message history from Bilibili.

## Requirements

- Python 3
- Google Chrome
- Selenium
- lxml

## Files

- [login.py](./login.py): opens Bilibili and records login data in a local Chrome profile.
- [message.py](./message.py): opens the Bilibili private message page and writes message history to a text file.
- [.gitignore](./.gitignore): ignores local personal data generated by this project.

## Usage

Run [login.py](./login.py) first:

```powershell
python login.py
```

After the browser opens, scan the QR code or finish login manually. The login profile will be saved here:

```text
selenium_chrome_profile/
```

Then run [message.py](./message.py):

```powershell
python message.py
```

If the script asks you to select a friend, select the friend in the browser window and wait for the script to continue.

## Output

The scraped message history is written to:

```text
msghistory/history.txt
```

## Sender Names

The output uses fixed sender labels in [message.py line 40](./message.py#L40) and [message.py line 42](./message.py#L42):

```python
sender = "you"
sender = "your friend"
```

Edit these strings if you want different names in `history.txt`.

## Notes

- `selenium_chrome_profile/` and `msghistory/` are local personal data folders. Do not commit login data or scraped private message history to GitHub.
